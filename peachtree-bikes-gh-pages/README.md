# Peachtree Bikes — Sales Platform Demo

Interactive sales platform prototype for Peachtree Bikes, built as a single-page React application.

## Pages

- **index.html** — Full sales platform demo (Pipeline, Deals, Leaderboard, Omni-channel, Workflows, Attribution, Price Watch)
- **prototype.html** — Product catalog prototype

## Deployment

This site is deployed via GitHub Pages. Push to `main` and enable Pages from Settings → Pages → Deploy from branch (`main`, root `/`).
